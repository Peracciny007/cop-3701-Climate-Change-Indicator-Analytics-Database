# Climate-Change-Indicator-Analytics-Database

The goal of this project is to design an environmental database tracking long term climate indicators. This dataset will include decades of data and analysis of their trends.
The data is sourced from Kaggle and it is from a public record of mean surface tempatures every year from 1961 to 2023. 
This dataset contains data from around 232 countries and it can be found herehttps://climatedata.imf.org/datasets/4063314923d74187be9596f10d034914_0/explore
This project will use professional documentation to anylize this dataset and and design relational schema.
